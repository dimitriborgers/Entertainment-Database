# 4111

## Setup 

1. Install python
`brew install python`

2. Install virtualenv 
`pip install virtualenv`

3. Clone the repo
`git clone https://github.com/Freeslugs/4111`

4. 
  - `cd 4111`
  - `. env/bin/activate`
  - `virtualenv env --python=python2.7`
  - `pip install -r requirements.txt`

## Run it 

1. `python app.py`
